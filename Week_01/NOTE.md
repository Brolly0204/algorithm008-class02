## 学习笔记

### 数据结构

#### Array 和 Linked List

| name       | prepend | appendd | lookup | insert | delete |
| ---------- | ------- | ------- | ------ | ------ | ------ |
| Array      | O(n)    | O(1)    | O(1)   | O(n)   | O(n)   |
| LinkedList | O(1)    | O(1)    | O(n)   | O(1)   | O(1)   |

#### Stack 和 Queue

- Stack 先入后出
- Queue 先入先出

Stack 的方法
method | description
-|-
empty()| 栈是否为空，返回 boolean 值
peek() | 返回栈顶元素，不会删除
push() | 向栈顶添加元素，入栈
pop() | 删除栈顶元素，出栈
search(E) | 返回元素所在位置

Queue 的方法
method | description
-|-
offer() | 队列中添加排队元素
poll() | 删除队列头部元素
peek() | 返回队列头部元素，不删除

Deque 双端队列还需要自己补补
